# Featured Project：项目笔记

`Featured Project` 是我当前公开展示的项目：一个可独立部署的国风田园经营项目。

它把单人经营冒险与账号、云存档、交流大厅、游戏邮箱、AI 小助理等在线能力结合在一起。

## 产品方向

这个项目处在经营游戏和 Web 应用之间。界面既要清晰耐用，也要保留田园世界需要的氛围。

我重点关注这些部分：

- 账号和存档流程要稳定清楚。
- 在线能力不能破坏单人体验。
- 国风田园表达要服务于可用性，而不是遮挡操作。
- 部署结构要尽量可复用、可独立运行。

## 工程记录

公开仓库主要由 TypeScript 和 Vue 构成，同时包含 JavaScript、Python、Java、CSS、HTML 和 Docker 相关内容。

这类项目的难点不是某个单独组件，而是让玩家数据、在线交互、界面表现、部署和长期维护保持一致。
